Learn to identify usability principles in real interface scenarios

In this game you will analyze different interface scenarios and determine which usability principle applies.

Usability Principles

Learnability

How easily users can learn to use the interface

Efficiency

How quickly users can perform tasks once they learn

Memorability

How easily users can remember how to use the interface

Error Prevention

How well the interface prevents user errors

User Satisfaction

How pleasant and satisfying the interface is to use
